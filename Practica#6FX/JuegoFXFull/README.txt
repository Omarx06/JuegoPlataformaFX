El juego consiste en un juego de Donkey Kong en el que vas subiendo a plataformas hasta llegar a la cima.

Estructura:
- src/: archivos .java de las clases
- assets/images: imágenes usadas para el juego (player, donkey, item, barril)
- datos/progreso.txt: archivo de persistencia usado por la clase Game

Como compilar y ejecutar
Instala Java 11+ y JavaFX SDK.
2. Compila:
Dar a Run para comenzar con la ejecución del código

Controles:
W A S D / flechas arriba, abajo, izquierda y derecha: mover el personaje
Barra espaciadora: saltar
Q: para guardar progreso
R: reiniciar
